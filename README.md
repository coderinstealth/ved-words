# Long A Words 📚

A fun phonics app for kids learning long A words.

## Deploy in 2 minutes (Vercel — free)

1. Install Node.js from https://nodejs.org if you don't have it
2. Install Vercel CLI:
   ```
   npm install -g vercel
   ```
3. In this folder, run:
   ```
   npm install
   vercel
   ```
4. Follow the prompts — it will give you a live URL like `https://long-a-words.vercel.app`

## Deploy with Netlify (also free)

1. Run:
   ```
   npm install
   npm run build
   ```
2. Go to https://netlify.com
3. Drag and drop the `dist` folder onto the Netlify dashboard
4. Done — instant live URL!

## Run locally

```
npm install
npm run dev
```
Then open http://localhost:5173
